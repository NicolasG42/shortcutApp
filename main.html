Frequently Asked Questions

Do I need admin rights?

Only for Fast Mode because it uses Windows Task Scheduler. Standard Mode works without admin rights via the Startup folder.

Can I assign multiple hotkeys per target?

Yes. Conflicts are detected and shown clearly.

Will my settings survive updates?

Yes. The configuration in AppData remains as long as you don’t reset it.

Is there a portable version?

Planned. For now, the regular installer is recommended to set up startup & shortcuts cleanly.

How to uninstall correctly?

Before uninstalling, please disable Fast Mode in settings so no scheduled tasks remain. Then remove via “Apps & Features”.
